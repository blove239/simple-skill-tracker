# simple-skill-tracker

In development gamified skill tracker to track progression of various skills. 

## Installation

```
git clone https://github.com/blove239/simple-skill-tracker.git
cd simple-skill-tracker
npm install
```

## Development

```
npm start
```

## Deployment

```
npm run deploy
```
